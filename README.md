## Installation

```bash
# 1. Clone into the right place
git clone https://github.com/meysam-kazemi/light-nvim-config.git ~/.config/nvim

# 2. Open Neovim
nvim

# 3. Let Lazy install everything
#    (just wait 10 seconds)
```

First start → type `:Lazy sync` → restart → DONE!

## Key Cheatsheet

Key,Action
Space e,Toggle left file tree
cmm,Comment line(s)
Space s,Toggle spell check
gt / gT,Next / previous buffer
